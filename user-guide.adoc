== Примеры кода

Ниже приведен пример инициализации клиента на Python:

[source,python]
----
import requests

def get_weather(city):
    url = f"https://api.openweathermap.org/data/2.5/weather?q={city}&appid={API_KEY}"
    response = requests.get(url)
    return response.json()
----
