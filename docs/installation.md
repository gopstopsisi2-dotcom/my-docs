# Руководство по установке

## Системные требования

| Компонент | Минимальные требования |
|-----------|------------------------|
| ОС | Windows 10 / macOS 11 / Ubuntu 20.04 |
| RAM | 2 GB |
| Диск | 500 MB свободного места |
| Python | 3.8 или выше |

## Установка из исходников

```bash
git clone https://github.com/gopstopsisi2-dotcom/my-docs.git
cd my-docs
pip install -r requirements.txt
python setup.py install

\# Установка программы



\## Скачивание



1\. Перейдите на сайт: https://example.com/download

2\. Нажмите кнопку "Скачать"

3\. Выберите версию для Windows



\## Установка



1\. Запустите скачанный файл `setup.exe`

2\. Нажмите "Далее"

3\. Примите лицензионное соглашение

4\. Выберите папку для установки

5\. Нажмите "Установить"

6\. Дождитесь завершения



\## Проверка установки



После установки запустите программу. Вы должны увидеть главное окно.
