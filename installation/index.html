# Установка программы

## Скачивание

1. Перейдите на сайт: https://example.com/download

2. Нажмите кнопку "Скачать"

3. Выберите версию для Windows

## Установка

1. Запустите скачанный файл setup.exe

2. Нажмите "Далее"

3. Примите лицензионное соглашение

4. Выберите папку для установки

5. Нажмите "Установить"

6. Дождитесь завершения

## Проверка установки

После установки запустите программу. Вы должны увидеть главное окно.
